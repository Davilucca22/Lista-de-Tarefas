# TaskFlow – Gerenciador Inteligente de Tarefas

Aplicação moderna de lista de tarefas construída com **React (Vite)** e **Tailwind CSS**, com **persistência via `localStorage`**.

## Funcionalidades

- **Adicionar tarefa**
- **Editar tarefa**
- **Excluir tarefa**
- **Persistência** com `localStorage`
- **Prioridade**: alta, média, baixa
- **Status**: pendente, concluída
- **UI moderna**: responsiva, com hover e transições

## Estrutura do projeto

```
src/
 ├── components/
 ├── pages/
 ├── hooks/
 ├── services/
 ├── styles/
```

## Como rodar

Instale as dependências:

```bash
npm install
```

Rode o servidor de desenvolvimento:

```bash
npm run dev
```
